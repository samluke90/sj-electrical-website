Replace `background: var(--dark);` with `background: #111111;`

The legacy `--dark` token maps to white text color in dark mode (confusing naming). Use explicit #111 as specified in requirements.

```css

}

    background: var(--color-bg-secondary);
    border-color: var(--color-bg-secondary);
}
```
